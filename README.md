# M-ENAS: Evolutionary neural architecture search for the automatic diagnosis of MDD using multi-modal MRI imaging
This repository contains the code and data for the paper "M-ENAS: Evolutionary neural architecture search for the automatic diagnosis of MDD using multi-modal MRI imaging". The project explores the integration of multimodal MRI data (sMRI, DTI, fMRI) using a novel neural architecture search method to improve diagnostic performance for MDD.
![Image](https://github.com/TTLi1996/M-ENAS/blob/main/M-ENAS.jpg)

# Summary
Major depressive disorder (MDD) is a prevalent mental disorder that seriously threatens human life and health. Multiple neuroimaging techniques can provide quantifiable evidence from different perspectives in the diagnosis of MDD. However, existing computer-aided diagnosis models for MDD are typically designed relying on the experience and intuition of researchers, which is time-consuming and may also make it difficult to obtain the globally optimal network architecture. To address this issue, we proposed an Evolutionary Neural Architecture Search (M-ENAS) framework for the automatic diagnosis of MDD using multi-modal MRI imaging. M-ENAS obtains the optimal weights and network architecture using a two-stage architecture search approach that allows selection from coarse to fine. Concretely, a One-shot NAS strategy was employed to train supernet weights during the supernet training stage, a self-defined evolutionary learning search strategy was used in the stage of exploring the optimal network architecture. Moreover, multi-modal neuroimaging features were integrated and recognized through feature concatenation and a fully connected layers (FC) with softmax activation function. Lastly, we evaluated the performance of M-ENAS on a set of clinical datasets, achieving an accuracy and F1-score of 75.00%, 0.7692, respectively, outperforming manually designed deep learning models and state-of-the-art NAS model.
